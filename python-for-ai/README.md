## 🔍 Key Insights Discovered

### Titanic Dataset
- Only **38%** of passengers survived
- Female survival rate was **2x higher** than male (women & children first!)
- `Cabin` column had the most missing values (687 out of 891)

### Student Performance Dataset
- Exam scores follow a **normal distribution** (bell curve, peak at 65–70)
- **Hours studied** is the #1 factor affecting exam scores
- Motivation level had surprisingly **little impact** on scores
